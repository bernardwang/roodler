Random + Doodle = Roodler

Roodler lets you see and post doodles on the Internet. Each time someone loads this website, they are presented with the most recently submitted doodle. The user then has the option to submit a new drawing and discard the old one. Roodler lets you make a random doodle for whoever opens Roodler next!

Think Snapchat, Chatroulette, and MS Paint combined! on that note, user discretion is advised
