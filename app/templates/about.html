Random + Doodle = Roodler

Roodler is a web application that allows you to see and post doodles on the internet. Each time someone loads the website they are presented with a new doodle that is viewable only once. The user then has the option to make a new doodle to replace the old one. Roodler lets you make a drawing for whoever opens Roodler next!

Basically, think Snapchat, Chatroulette, and Draw My Thing combined. similarly, user discretion is advised

Roodler was made by Bernard Wang and Joe Higgs
